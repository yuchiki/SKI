## how to install

- install [stack](https://docs.haskellstack.org/en/stable/README/)
- download this repository
- `stack install` in this repository

## manual

    ?                         : show help
    :s                        : show definitions
    :v                        : flip verbosity
    <term>                    : evaluate term
    import <library>          : import library
    let <identifier> = <term> : define term

Type `?` in SKI Repl.
